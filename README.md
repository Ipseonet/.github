# .github

Organization-wide defaults for the **IPSEONET** GitHub organization.

Files here apply to every repository in the org that doesn't provide its own:

- **`SECURITY.md`** — default security / vulnerability-reporting policy.
- **`profile/README.md`** — the public organization profile shown at
  [github.com/ipseonet](https://github.com/ipseonet).

To override for a specific repository, add that file directly to the repo.

> This repository must remain **public** for the default files to take effect.
